# GK1200 维修手册查询 Skill

高金 **GK1200**（手册内亦称 BX500 / BX1200）摩托车维修手册的 Claude Code 查询 skill。
一问扭矩、保养周期、油液规格、拆装步骤即答；支持识图的 agent 还能直接看整页图解。

## 这是什么

一个面向 Claude Code（及兼容 skill 机制的 agent）的本地 skill：

- 把整本维修手册做成可被 agent 自动检索的结构化资料
- 常查的硬数据（扭矩 / 保养 / 油液 / 整车参数 / 火花塞 / 胎压 / 缸压 / 电气）内置成「快查表」，秒答
- 细节问题按主题索引在全文 `grep` 定位
- 涉及图解时，整页图（每页一张、无碎片）按需提供

## 目录结构

```
gk1200-repair/
├─ SKILL.md     # skill 入口：触发描述 + 快查表 + 主题索引 + 看图规则
├─ manual.md    # 手册全文（已清洗），图链接指向整页图
└─ pages/       # 184 页整页图（WebP）
```

## 安装

1. 把本仓库克隆或复制到 Claude Code 的用户级 skill 目录，使其最终位于 `<skills>/gk1200-repair/`：
   - macOS / Linux：`~/.claude/skills/gk1200-repair/`
   - Windows：`%USERPROFILE%\.claude\skills\gk1200-repair\`
2. 重启 Claude Code（新建会话）使其加载 skill。
3. 直接提问即可触发，例如：「GK1200 后轮轴螺母扭矩多少？」

> skill 内引用自身文件均采用「相对本 skill 基目录」的写法，因此克隆到任意路径都可用。

## 用法示例

- GK1200 后轮轴螺母扭矩多少？
- 机油多久换一次、加多少？
- 火花塞什么型号、间隙多少？
- 节气门体怎么拆？（给步骤 + 对应整页图；支持识图的 agent 会直接解读图）

## 看图：两类 agent 都兼容

- **不具识图能力的 agent / 你本人**：skill 给出整页图路径 + 图旁说明文字，自行打开查看。
- **具备识图能力的 agent**：直接读取整页图并结合问题解读（部件、拆装顺序、走线等）。

## ⚠️ 安全提示

扭矩值为安全数据。本手册同一部位在不同章节存在**不一致**的扭矩值，skill 会把多个来源值并列、提示以原书为准并取保守做法。**实际作业请以纸质 / 官方手册为最终依据。**
